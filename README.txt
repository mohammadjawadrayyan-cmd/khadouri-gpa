# حاسبة المعدل - جامعة فلسطين التقنية – خضوري (v2)

## التشغيل (بدون أي أدوات)
1) افتح الملف `index.html` بالمتصفح مباشرة (Double Click).
2) أدخل مواد الفصل -> احسب المعدل الفصلي.
3) في قسم التراكمي: أدخل المعدل والساعات السابقة -> استخدم نتيجة الفصل الحالي -> احسب المعدل التراكمي.

## مواد مُعادة (اختياري)
- استخدمها فقط إذا العلامة القديمة للمساق ما زالت محسوبة ضمن المعدل التراكمي السابق الذي أدخلته.
- اختر السياسة:
  - "استبدال": سيطرح أثر العلامة القديمة (العلامة×الساعات) من الرصيد السابق ثم يضيف علامة هذا الفصل.
  - "احتساب المحاولتين": لن يطرح شيء (يعني القديم يبقى + الجديد يضاف).

## تشغيل عبر سيرفر محلي (اختياري)
python -m http.server 8000
ثم افتح:
http://localhost:8000
